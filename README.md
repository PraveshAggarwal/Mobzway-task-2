# Babu88 Batting Site (Replica)

A front-end replica of the **Babu88 Batting** website UI/experience, built as part of **Mobzway Task 2**.  
This project focuses on recreating the look and feel of the original site: layout, styling, sections, and interactions.

## Live Demo

Hosted on Render:  
https://mobzway-task-2-zr5k.onrender.com

## Project Overview

This repository contains a JavaScript-heavy web implementation that replicates the core UI flow of the Babu88 batting site. It is primarily intended for:

- Practicing pixel-accurate UI replication
- Working with DOM manipulation and client-side interactions
- Rebuilding real-world website sections with HTML/CSS/JavaScript

## Features

- Replica landing page layout similar to the original Babu88 batting site
- Responsive styling for common screen sizes
- Interactive UI elements (buttons/menus/sections) handled via JavaScript
- Clean separation of structure (HTML), styling (CSS), and behavior (JavaScript)

## Tech Stack

- **JavaScript** (primary)
- **CSS**
- **HTML**

## Getting Started (Run Locally)

### 1) Clone the repository
```bash
git clone https://github.com/PraveshAggarwal/Mobzway-task-2.git
cd Mobzway-task-2
```

### 2) Run the project
This project is a static front-end app. You can run it using any static server.

#### Option A: Using VS Code Live Server
1. Open the folder in VS Code
2. Install **Live Server** extension (if not installed)
3. Right-click `index.html` → **Open with Live Server**

#### Option B: Using `http-server`
```bash
npm install -g http-server
http-server
```
Then open the URL shown in the terminal (commonly `http://localhost:8080`).
